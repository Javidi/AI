---------------------------------------------------------------------------------

General Information:

In order to run this script, **you will need approximately 2.4 Gigabytes of free storage space and memory (RAM)**.

If your local computer does not have adequite storage or memory, please utilize the university cloud computing server: https://cloudgw.cpcc.edu/

---------------------------------------------------------------------------------

Steps for Windows: 

1. run the command to initialize: ```. .\windows_setup.ps1```
2. look inside the 'chat.py' file for details of the code you will be running
3. run the command to open the chat interface: ```python chat.py```

---------------------------------------------------------------------------------

Steps for Mac/Linux: 

1. run the command to initialize: ```source mac_setup.sh```
2. look inside the 'chat.py' file for details of the code you will be running
3. run the command to open the chat interface: ```python3 chat.py```
